React Movies

R

Interacting with the Movie Data Base API

Brief:

A fun little app I built to interact with the Movie DB API. Similiar to IMDB functionality, but less robust. I used React Hooks, Functional Components, and StyledComponents. I had some trouble mapping over the returned arrays for a bit, but I was able to console.log the situation out to my custom useMovieFetch Hook. Once those fetch errors were corrected (you must map over the elemtn AND its index) the rest was down hill and I'm overall super stoked on the project.
