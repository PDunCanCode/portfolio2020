Neuron Social Network a full stack social media application to help cohortians stay connected.

Brief:

Neuron was my final project for Geekwise Academy's Web Development Bootcamp. I was the sole developer on this full stack MERN application. The interactive animated background is built on three.js, and the application is hosted live on heroku. The hardest bug I solved was working through authentication libraries. This was the first full stack application I had built without using Firebase for auth, opting for JWT instead, the learning curve was steep, but rewarding.

Client
Geekwise Academy
Technologies Used:

React, JWT, MondoDB, Express, Node.js, three.js
